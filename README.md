# spring-boot-and-react-with-netty-chat
비동기 고성능 네트워크 서버인 Netty를 공부하며 이해 하기 위해 간단한 채팅서버와 클라이언트를 만들어 보았습니다.
기초 모델은 간단한 현 상태로 유지하고, 점차 기능과 테스트 코드를 붙여가며 개선 할 계획입니다.

## 사용 기술 및 환경
Server: Spring boot
Client: React + Redux + TypeScript
